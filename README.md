# OLA-Sales-Analysis-Dashboard

# 🚖 Ola Sales Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze Ola ride booking data and provide actionable business insights into revenue, ride trends, cancellations, customer ratings, and vehicle performance.

---

## 📌 Project Overview

This dashboard helps stakeholders monitor key business metrics and identify patterns in ride bookings, revenue generation, cancellations, and customer satisfaction.

The dashboard is divided into multiple report pages for detailed analysis:

- 📊 Overall Dashboard
- 🚗 Vehicle Type Analysis
- 💰 Revenue Analysis
- ❌ Cancellation Analysis
- ⭐ Ratings Analysis

---

## 🎯 Objectives

- Analyze overall ride booking performance.
- Monitor revenue trends over time.
- Compare vehicle category performance.
- Identify major cancellation reasons.
- Evaluate customer and driver ratings.
- Enable business-driven decision making through interactive visualizations.

---

## 📈 Dashboard Features

### 📊 Overall Dashboard
- Total Bookings
- Total Revenue
- Ride Trend Analysis
- Booking Distribution
- Interactive Filters

### 🚗 Vehicle Type Analysis
- Vehicle-wise booking count
- Vehicle-wise revenue comparison
- Performance comparison across ride categories

### 💰 Revenue Dashboard
- Revenue trends
- Revenue by booking type
- Revenue breakdown tables
- Time-based revenue analysis

### ❌ Cancellation Dashboard
- Customer cancellations
- Driver cancellations
- Cancellation percentage
- Cancellation reason distribution

### ⭐ Ratings Dashboard
- Customer ratings
- Driver ratings
- Average rating KPIs
- Rating distribution

## 🚀 How to Use

1. Clone this repository.

```bash
git clone https://github.com/yourusername/Ola-Sales-Dashboard.git
```

2. Open the `.pbix` file using **Power BI Desktop**.

3. Refresh the dataset (if required).

4. Explore the interactive dashboard using slicers and navigation buttons.
